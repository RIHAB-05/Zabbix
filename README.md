# Zabbix
**l’installation et la configuration complète d’une plateforme de supervision Zabbix**

Ce projet fournit un ensemble de commandes permettant d’installer, configurer et mettre en service une solution complète de supervision basée sur Zabbix.
Il inclut la mise à jour de MariaDB vers la version 10.5, l’installation du Zabbix Server, de l’interface Web et l’initialisation de la base de données Zabbix avec import du schéma officiel.
Il couvre également l’installation et la configuration du Zabbix Agent.
La procédure est destinée aux systèmes Linux basés sur RHEL 8, Oracle Linux 8 ou CentOS Stream 8, qui utilisent le gestionnaire de paquets DNF et le système de modules.
